# 🌫️ Air Quality Trend Dashboard

A live Streamlit web app analysing AQI sensor data for 5 major Indian cities over 2 years.

## 🔗 Live Demo
> Deploy link appears here after Streamlit Cloud deployment

## 🛠 Skills Demonstrated
`Python` `Pandas` `Matplotlib` `Seaborn` `Streamlit` `Data Analytics`

## ✨ Features
- Interactive city & year filter in sidebar
- KPI cards — avg AQI, worst/best city, spike count
- 4 tabs: Trends · Heatmap · Seasonal · Pollutants
- IQR-based pollution spike detection
- Pollutant-level comparison (PM2.5, PM10, NO2, SO2, CO, O3)

## 🚀 Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## ☁️ Deploy to Streamlit Cloud
1. Push this folder to a GitHub repo
2. Go to share.streamlit.io → New app
3. Select repo → select `app.py` → Deploy
